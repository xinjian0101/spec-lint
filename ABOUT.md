# About Spec Lint

Spec Lint is a planned command-line checker for software requirement documents.

## Suggested GitHub About

Linter for software specifications, acceptance criteria, edge cases, data rules, and requirement consistency.

## Topics

`requirements`, `specifications`, `linter`, `developer-tools`, `documentation`, `cli`

## Purpose

The project will help users find missing or unclear information before implementation. Findings will reference documented rules and source locations.

## Current status

Foundation phase. The rule set and report formats are being designed.

## Maintenance

Rule changes should include examples and tests. User-visible changes will be recorded in release notes.
